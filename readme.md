# Algoritmos de ordenamiento

Proyecto educativo realizado en HTML, CSS y JavaScript.
Trata sobre los principales algoritmos de ordenamiento, una breve descripción de cada uno de ellos, como funcionan y sus caracteristicas técnicas.
Este proyecto prioriza la claridad conceptual sobre el rendimiento real y fue desarrollado con fines educativos como práctica de algoritmos, estructura de código y diseño de interfaces interactivas.

## IMPORTANTE

En la versión actual, los algoritmos no presentan diferencias visibles en el resultado final al ordenar un arreglo (más allá de sus características técnicas teóricas).
Sin embargo, cada algoritmo está implementado de forma independiente, ya que el proyecto está pensado para evolucionar en futuras versiones, incorporando diferencias visuales y métricas prácticas reales.
Este proyecto utiliza JavaScript con ES Modules (`import / export`).

Por este motivo **no funciona abriendo el archivo `index.html` directamente** (doble click),
ya que los navegadores bloquean los módulos cuando se ejecutan desde el sistema de archivos.

### Formas correctas de ejecutarlo:

- Usando **Live Server** en Visual Studio Code
- O accediendo a la versión publicada en **GitHub Pages**

Esto no es un error del proyecto, sino una característica del uso de módulos en JavaScript moderno.


## Funcionalidades

- Ingreso manual de elementos de un arreglo (números separados por comas)

- Selección interactiva del algoritmo de ordenamiento (Bubble Sort, Selection Sort, Insertion Sort, Quick Sort, Merge Sort)

- Ejecución del algoritmo seleccionado

- Visualización del arreglo ordenado

- Mostrar detalles técnicos (teóricos) del algoritmo:

- Complejidad temporal

- Cantidad de comparaciones (orden de magnitud)

- Cantidad de intercambios (orden de magnitud)

- Estabilidad

- Uso de memoria (in-place o no)

- Posibilidad de limpiar el resultado y volver a probar

## Cómo usar el proyecto

- Abrí el archivo index.html en el navegador

- Ingresá los números del arreglo separados por comas (Ejemplo: 7, 3, 9, 1, 5)

- Seleccioná uno de los algoritmos disponibles

- Presioná el botón Ordenar

- Observá el resultado y los detalles técnicos

- Usá Limpiar para ocultar el resultado y volver a probar
